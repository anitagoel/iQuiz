Publish

If you are done with creating the quiz, click Publish button on this page. This will make the quiz accessible to your students. You might also be required to click Publish button on the Studio for this Unit to put make the quiz live.


Please note that you will not be able to edit the questions of the quiz if any students attempts the quiz! So make sure that you are done with the quiz before publishing!

Delete Quiz

You can also delete the Quiz!

Please note that deleting the quiz will delete:

  • All the Questions of the Quiz
  • Settings of the Quiz
  • All the information associated with the Students who accessed/attempted the quiz
  • All the associated Response of the students.

Please be super sure before doing this!
Basically, this will make it like a new Quiz just initialized! Everything will be wiped out!
